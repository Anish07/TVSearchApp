# TV Show Search App

A simple, premium-styled web application that allows users to search for TV shows and view their posters. Built using the [TVMaze API](https://www.tvmaze.com/api).

## Features
- **Search Functionality**: Instantly find TV shows by title.
- **Premium UI**: Modern dark mode design with responsive grid layout and hover effects.
- **Dynamic Content**: Fetches and displays show posters dynamically.

## Technologies Used
- **HTML5**: Structure and semantics.
- **CSS3**: Custom styling with Flexbox and Grid.
- **JavaScript (ES6+)**: Logic and DOM manipulation.
- **Axios**: Handling API requests.

## How to Run
1. Clone the repository.
2. Open `index.html` in your web browser.
3. Start searching!

## Deployment
This project is deployment-ready for platforms like [Vercel](https://vercel.com) or Netlify.
